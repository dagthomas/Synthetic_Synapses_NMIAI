Rules & Scoring

Scoring Formula

score = 100 × exp(-3 × weighted_kl)
weighted_kl = Σ(entropy × KL) / Σ(entropy)
KL(p||q) = Σ pᵢ × log(pᵢ / qᵢ)

Only dynamic cells (entropy > 0.01) count. Static terrain (void, obsidian) is free.

Critical Rules

  • Never assign probability 0 — use floor of 0.01 minimum
  • Each cell: 6 probabilities must sum to 1.0 (±0.01)
  • 50 scans shared across 5 scan zones per planet
  • Each scan shows a different stochastic outcome
  • Resubmitting overwrites previous prediction
  • Missing seeds score 0 — always submit all 5

26 Hidden Planetary Physics

Each planet has unique physics you must infer from observations. These control everything:

  • base_survival — Crystal node persistence
  • expansion_str — Growth rate strength
  • expansion_scale — Growth distance decay
  • decay_power — Distance falloff sharpness
  • max_reach — Maximum growth radius
  • coastal_mod — Void-edge survival modifier
  • food_coeff — Xenoflora nutrition boost
  • cluster_pen — Overcrowding penalty
  • cluster_optimal — Ideal neighbor density
  • cluster_quad — Density penalty curve
  • ruin_rate — Collapse → depletion rate
  • port_factor — Refinery formation chance
  • forest_resist — Xenoflora blocks growth
  • forest_clear — Xenoflora clearing rate
  • forest_reclaim — Surface reclamation rate
  • exp_death — New crystal death chance
  • neighbor_survival_bonus — Faction cooperation
  • faction_kill_prob — Faction conflict lethality
  • faction_range — Conflict engagement range
  • forest_survival_bonus — Xenoflora protection
  • forest_repro_bonus — Xenoflora growth boost
  • ruin_to_settlement_prob — Vein reclamation
  • port_upgrade_prob — Refinery upgrade rate
  • port_spawn_multiplier — Refinery growth bonus
  • winter_severity — Periodic die-off severity
  • trade_bonus — Inter-refinery trade boost
