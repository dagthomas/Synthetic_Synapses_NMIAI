# Astar Island — Learnings

This folder contains discovered mechanics and patterns from exploring the simulation.

Files are updated incrementally as more rounds are observed.

## Index

- `terrain_transitions.md` — What terrain types become what (with observed frequencies)
- `settlement_dynamics.md` — Survival rates, expansion patterns, ruin rates
- `port_and_trade.md` — Port formation patterns, trade effects
- `conflict_patterns.md` — Raid outcomes, faction changes
- `spatial_patterns.md` — How far settlements expand, clustering effects
- `cross_seed_variance.md` — How much outcomes vary across seeds
- `scoring_insights.md` — Which cells have highest entropy weight, where predictions matter most
